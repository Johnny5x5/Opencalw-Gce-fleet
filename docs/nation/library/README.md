# The National Library of the Sovereign Digital Nation

**Status:** Under Construction
**Custodian:** The Librarian

## Structure
- **[halls/](./halls/)**: The Main Halls. Categorized, verified, and active knowledge.
- **[archives/](./archives/)**: The Vault. Historical records, immutable and deep-frozen.
- **[shacks/](./shacks/)**: The Outskirts. Orphaned, unverified, or temporary knowledge waiting for classification.
