# Data Privacy Policy

**1. Data Classification**
*   **Public:** Marketing materials, Open Source code.
*   **Internal:** Memos, non-sensitive Slack chat.
*   **Confidential (CUI):** Source code, Strategy docs. (Requires Level 2 Security).
*   **Restricted (PII/PHI):** Customer data, Financial records. (Requires Level 3/4 Security).

**2. Storage Requirements**
*   Restricted data must rely on **Confidential VMs** (Data-in-Use protection).
*   All data buckets must use **CMEK** (Customer Managed Encryption Keys).

**3. Retention**
*   Financial Logs: 7 Years (WORM Locked).
*   Call Transcripts: 1 Year (Coldline).
*   Ephemeral Chat: 30 Days.

**4. Access Control**
*   Access to Restricted data requires **Two-Person Rule** (Human + Agent approval).
