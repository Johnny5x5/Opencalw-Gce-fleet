# OpenClaw: The Sovereign Digital Nation

## The Grand Architecture
This repository contains the blueprints and implementation for a Sovereign Digital Nation.

### Core Systems (The Body)
*   **[The Iron Ledger (SBOM)](packages/sbom-secure/):** Military-Grade Software Bill of Materials (DoD IL5).
*   **[The National Library (Brain)](packages/national-library/):** AI-First Knowledge Graph containing the [Constitution](docs/nation/CONSTITUTION_OF_THE_REPUBLIC.md) and [Sacred Texts](docs/nation/library/sacred_texts/).
*   **[The Mobius Engine (Nervous System)](packages/mobius-engine/):** Federated API Gateway linking Social Groups to Governance.

### Governance (The Mind)
*   **[The War Council](docs/departments/WAR_COUNCIL.md):** Strategic Defense and Verification.
*   **[The Chaplaincy](docs/departments/CHAPLAINCY.md):** Moral Audit and 10% Resource Cap.
*   **[The Scribes](docs/nation/protocols/COUNCIL_INTERACTION.md):** Historical, Legal, and Cultural Memory.
